Data types

single variable:

int – Integer (e.g., 10)
float – Decimal number (e.g., 3.5)
complex – Complex number (e.g., 2+3j)
str – String/text (e.g., "Hello")
multi variable:

list – Ordered, changeable (e.g., [1, 2, 3])
tuple – Ordered, not changeable (e.g., (1, 2, 3))
range – Sequence of numbers (e.g., range(5))
set – Unordered, no duplicates (e.g., {1, 2, 3})
dict – Key-value pairs (e.g., {"a": 1})


OPERATORS

1. Arithmetic operators
+(Addition) → 5 + 2 = 7
- (Subtraction) → 5 - 2 = 3
* (Multiplication) → 5 * 2 = 10
/ (Division) → 5 / 2 = 2.5
% (Modulus) → 5 % 2 = 1
* (Power) → 2 * 3 = 8
// (Floor division) → 5 // 2 = 2


2. Comparison Operators
Used to compare values (result is True/False)
== (Equal)
!= (Not equal)
> (Greater than)
< (Less than)
>= (Greater or equal)
<= (Less or equal)


3. Logical Operators
Used with conditions
and → True if both are true
or → True if at least one is true
not → Reverses the result


4. Assignment Operators
Used to assign values
=
+=, 
-=, 
 =,
   /=,
 %=,
 //=,
  *=


5. Bitwise Operators
Work on binary values
 
 &,
 |,
 ^,
 ~,
 <<,
 >>


6. Membership Operators
Check if value exists in a sequence
in
not in


7. Identity Operators
Check if two variables refer to same object
is
is not
